1. Introduction au HTML

HTML (HyperText Markup Language) est le langage fondamental du web. Créé par Tim Berners-Lee en 1989, il permet de structurer le contenu des pages web.

Pourquoi le HTML est-il important ?

  • Il fournit la structure de base pour toutes les pages web
  • Il est compatible avec tous les navigateurs
  • Il est la base pour d'autres technologies web comme CSS et JavaScript

Évolution du HTML

Le HTML a évolué au fil des années :

  • HTML 1.0 (1991) : Version initiale
  • HTML 4.01 (1999) : Ajout de feuilles de style et scripts
  • XHTML (2000) : Version plus stricte basée sur XML
  • HTML5 (2014) : Introduction d'éléments sémantiques et multimédia

Exemple : Structure de base HTML5

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ma première page</title>
</head>
<body>
    <h1>Bienvenue sur ma page</h1>
    <p>Ceci est un paragraphe.</p>
</body>
</html>

2. Structure et Sémantique HTML5

HTML5 a introduit de nombreux éléments sémantiques qui donnent plus de sens à la structure de votre page. Ces éléments aident les moteurs de recherche et les technologies d'assistance à mieux comprendre votre contenu.

Principaux éléments sémantiques HTML5

  • <header> : En-tête de la page ou d'une section
  • <nav> : Section de navigation
  • <main> : Contenu principal de la page
  • <article> : Contenu autonome et indépendant
  • <section> : Section thématique du contenu
  • <aside> : Contenu connexe ou secondaire
  • <footer> : Pied de page

Exemple : Structure sémantique HTML5

<body>
    <header>
        <h1>Mon Site Web</h1>
        <nav>
            <ul>
                <li><a href="#">Accueil</a></li>
                <li><a href="#">À propos</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <article>
            <h2>Titre de l'article</h2>
            <p>Contenu de l'article...</p>
        </article>

        <aside>
            <h3>Contenu connexe</h3>
            <p>Informations supplémentaires...</p>
        </aside>
    </main>

    <footer>
        <p>&copy; 2023 Mon Site Web. Tous droits réservés.</p>
    </footer>
</body>

3. Les éléments de texte

Les éléments de texte en HTML permettent de structurer et de formater le contenu textuel d'une page web. Ils incluent les titres, paragraphes, citations, et balises de mise en forme.

Principaux éléments de texte

  • <h1> à <h6> : Titres de différents niveaux
  • <p> : Paragraphe
  • <strong> : Texte important (généralement en gras)
  • <em> : Texte mis en emphase (généralement en italique)
  • <blockquote> : Citation longue
  • <q> : Citation courte
  • <abbr> : Abréviation

Exemple : Éléments de texte

<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<p>Ceci est un <strong>paragraphe</strong> avec du texte <em>mis en valeur</em>.</p>
<blockquote>
    Ceci est une longue citation qui peut s'étendre sur plusieurs lignes.
</blockquote>
<p>Comme l'a dit quelqu'un, <q>Une citation courte</q>.</p>
<p>L'<abbr title="Organisation des Nations Unies">ONU</abbr> a été fondée en 1945.</p>

4. Les liens hypertextes

Les liens hypertextes sont essentiels pour la navigation sur le web. Ils permettent de connecter différentes pages ou ressources entre elles.

Types de liens

  • Liens externes (vers d'autres sites web)
  • Liens internes (vers d'autres pages du même site)
  • Liens d'ancrage (vers une section spécifique de la même page)
  • Liens pour e-mail et téléphone

Exemple : Liens hypertextes

<a href="https://www.example.com">Lien externe</a>
<a href="/page.html">Lien interne</a>
<a href="#section1">Lien vers une ancre</a>
<a href="mailto:contact@example.com">Envoyer un email</a>
<a href="tel:+33123456789">Appeler</a>

5. Les images et médias

L'intégration d'images et de médias enrichit le contenu d'une page web. HTML5 offre des balises natives pour l'audio et la vidéo.

Éléments multimédias

  • <img> : Pour les images
  • <audio> : Pour l'audio
  • <video> : Pour la vidéo
  • <figure> et <figcaption> : Pour les images avec légende

Exemple : Images et médias

<img src="https://via.placeholder.com/150" alt="Image d'exemple">

<figure>
    <img src="https://via.placeholder.com/300x200" alt="Une belle image">
    <figcaption>Légende de l'image</figcaption>
</figure>

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Votre navigateur ne supporte pas l'élément audio.
</audio>

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Votre navigateur ne supporte pas l'élément vidéo.
</video>

6. Les listes

Les listes en HTML permettent d'organiser et de structurer l'information. Il existe trois types principaux de listes : ordonnées, non ordonnées, et de définition.

Types de listes

  • Liste non ordonnée (<ul>)
  • Liste ordonnée (<ol>)
  • Liste de définition (<dl>)

Exemple : Listes

<h3>Liste non ordonnée</h3>
<ul>
    <li>Pommes</li>
    <li>Bananes</li>
    <li>Oranges</li>
</ul>

<h3>Liste ordonnée</h3>
<ol>
    <li>Première étape</li>
    <li>Deuxième étape</li>
    <li>Troisième étape</li>
</ol>

<h3>Liste de définition</h3>
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets</dd>
</dl>

  3. Troisième étape

7. Les tableaux

Les tableaux en HTML sont utilisés pour présenter des données sous forme de lignes et de colonnes. Ils sont composés de plusieurs éléments comme table, tr, td, th, etc.

Structure d'un tableau

  • <table> : Définit un tableau
  • <tr> : Définit une ligne
  • <th> : Définit une cellule d'en-tête
  • <td> : Définit une cellule de données
  • <caption> : Ajoute un titre au tableau

Exemple : Tableaux

<table>
    <caption>Exemple de tableau</caption>
    <thead>
        <tr>
            <th>En-tête 1</th>
            <th>En-tête 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Donnée 1</td>
            <td>Donnée 2</td>
        </tr>
        <tr>
            <td>Donnée 3</td>
            <td>Donnée 4</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">Pied de tableau</td>
        </tr>
    </tfoot>
</table>

8. Les formulaires

Les formulaires HTML permettent aux utilisateurs d'interagir avec un site web en saisissant des données. Ils sont essentiels pour la collecte d'informations et l'interaction utilisateur.

Éléments de formulaire

  • <form> : Conteneur du formulaire
  • <input> : Champ de saisie (texte, case à cocher, bouton radio, etc.)
  • <textarea> : Zone de texte multiligne
  • <select> et <option> : Liste déroulante
  • <button> : Bouton
  • <label> : Étiquette pour les champs de formulaire

Exemple : Formulaires

<form action="/submit-form" method="post">
    <label for="name">Nom :</label>
    <input type="text" id="name" name="name" required>

    <label for="email">Email :</label>
    <input type="email" id="email" name="email" required>

    <label for="password">Mot de passe :</label>
    <input type="password" id="password" name="password" required>

    <label for="country">Pays :</label>
    <select id="country" name="country">
        <option value="france">France</option>
        <option value="canada">Canada</option>
        <option value="belgique">Belgique</option>
    </select>

    <fieldset>
        <legend>Préférences :</legend>
        <input type="checkbox" id="newsletter" name="newsletter">
        <label for="newsletter">S'abonner à la newsletter</label>
    </fieldset>

    <button type="submit">Envoyer</button>
</form>

9. Les éléments de mise en page

Les éléments de mise en page en HTML5 permettent de structurer le contenu de manière sémantique. Ils aident à organiser le contenu et améliorent l'accessibilité et le SEO.

Éléments de mise en page HTML5

  • <header> : En-tête de la page ou d'une section
  • <nav> : Navigation principale
  • <main> : Contenu principal de la page
  • <article> : Contenu autonome
  • <section> : Section thématique de contenu
  • <aside> : Contenu connexe ou secondaire
  • <footer> : Pied de page

Exemple : Éléments de mise en page

<header>
    <h1>Mon Site Web</h1>
    <nav>
        <ul>
            <li><a href="#home">Accueil</a></li>
            <li><a href="#about">À propos</a></li>
            <li><a href="#contact">Contact</a></li>
        </ul>
    </nav>
</header>

<main>
    <article>
        <h2>Article Principal</h2>
        <p>Contenu de l'article...</p>
    </article>
    <aside>
        <h3>Contenu Annexe</h3>
        <p>Informations supplémentaires...</p>
    </aside>
</main>

<footer>
    <p>&copy; 2024 Mon Site Web. Tous droits réservés.</p>

10. Validation et bonnes pratiques

La validation du code HTML et le respect des bonnes pratiques sont essentiels pour créer des sites web de qualité. Cela améliore l'accessibilité, le SEO et la compatibilité entre les navigateurs.

Bonnes pratiques HTML

  • Utiliser la déclaration DOCTYPE appropriée
  • Spécifier la langue du document
  • Utiliser des balises sémantiques
  • Assurer l'accessibilité (textes alternatifs, structure logique)
  • Valider le code HTML

Exemple : Validation HTML

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Exemple</title>
</head>
<body>
    <h1>Titre Principal</h1>
    <p>Un paragraphe avec un <a href="https://example.com">lien</a>.</p>
    <img src="https://placehold.co/200x200" alt="Description de l'image">
</body>
</html>
